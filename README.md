# Telegram Bot for Creating Notes

A bot built with aiogram that collects notes from notebook photos and generates short summaries.
Test it: [https://t.me/conspectssss_bot](https://t.me/conspectssss_bot)

## Features

* Accepts photos of notebook pages
* Automatic text recognition (OCR) using Google Cloud Vision API or Tesseract
* Combines text from multiple photos into one note
* Generates a short summary using OpenAI GPT

## Usage

1. Send the `/start` command to the bot
2. Send photos of your notebook pages
3. After uploading all photos, press the **“Summarize”** button
4. The bot will generate and send a brief summary of the notes
5. Use `/clear` to clear the current note


# Telegram Bot для создания конспектов

Бот на aiogram, который собирает конспекты из фотографий тетради и генерирует краткие резюме.
Протестировать: https://t.me/conspectssss_bot

## Возможности

- Прием фотографий из тетради
- Автоматическое распознавание текста (OCR) с помощью Google Cloud Vision API или Tesseract
- Сбор конспекта из нескольких фото
- Генерация краткого резюме с помощью OpenAI GPT


## Использование

1. Отправьте команду `/start` боту
2. Отправляйте фотографии из тетради
3. После добавления всех фото нажмите кнопку "Резюмировать"
4. Бот сгенерирует и отправит краткое резюме конспекта
5. Используйте `/clear` для очистки текущего конспекта
